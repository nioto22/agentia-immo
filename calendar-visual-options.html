Recapitulatif
Option Style Forces Limites
A — Grille Calendrier classique Vue d'ensemble semaine, familier Texte petit sur mobile, dense
B — Timeline Fil chronologique Tres lisible, mobile-friendly, narratif Plus long a scroller
C — Kanban Colonnes plateforme Vue par canal, actionnable Perd la chronologie, 2 colonnes max
D — Agenda Google Calendar Ultra compact, overview rapide Trop petit pour details, pas mobile
E — Cartes Magazine / premium Le plus visuel, premium, lisible Prend plus de place verticalement
